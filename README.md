## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IbrahimShafiq4/foody.git
   ```
2. Navigate to the project directory:
   ```bash
   cd foody
   ```
3. Install dependencies using npm:
   ```bash
   npm install
   ```
   This will install all required dependencies, including:
   - **jwt-decode**: For decoding JWT tokens.
   - **Font Awesome**: For scalable vector icons.
   - **Bootstrap**: For responsive and mobile-first front-end web development.
   - **Angular Material**: For implementing Material Design components in Angular applications.
   
4. After installation, you may need to configure additional settings based on your environment and project requirements.

5. Follow any additional setup instructions provided in the project's README file to configure Foody according to your needs.

## Usage
Explore Foody to streamline food-related tasks and operations. Users can manage their recipes, ingredients, and meal plans, while administrators have access to additional features for managing users, menus, and orders. With Angular's robust framework, Foody offers a responsive and intuitive experience for all users.
